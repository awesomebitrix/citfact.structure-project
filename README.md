Cтруктура проекта
===

Скелет проекта для 1C-Bitrix

## Требования:

 - PHP версия >= 5.3.3
 - Bitrix версия >= 14

## Установка

``` bash
# Переходим в папку проекта
cd /to/path/project/

# Клонируем репозиторий
git init .
git remote add -t \* -f origin https://github.com/studiofact/citfact.structure-project.git
git checkout master
```
